# QA Evaluation

This is a Python module for assessing the quality of agents such as ones based on LLMs, based on a set of questions and reference answers for them. This includes evaluating the final answer and the steps used to reach the answer (such as orchestrated and executed steps), compared to the given reference steps. The library provides built-in evaluation metrics and allows the user to define their own (custom) metrics. The library is agnostic to the agent implementation and LLM it uses. Its input format is versatile and expressive.
